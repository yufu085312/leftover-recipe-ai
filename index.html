ステップ1: Google AI Studioにアクセス

🔗 Google AI Studio を開いてください

ステップ2: Googleアカウントでログイン

お持ちのGoogleアカウントでログインしてください

ステップ3: APIキーを作成

  • 「Create API key」または「Get API key」ボタンをクリック
  • 「Create API key in new project」を選択（推奨）
  • APIキーが生成されます（AIza...で始まる文字列）

ステップ4: APIキーをコピー

表示されたAPIキーをコピーして、上記の入力欄に貼り付けてください

💡 重要: APIキーは無料枠があり、1日1,500リクエストまで利用可能です

独自のAPIキーはUnsplash Developersで取得できます（無料）

ステップ1: Unsplash Developersにアクセス

🔗 Unsplash Developers を開いてください

ステップ2: アカウント登録・ログイン

「Register as a developer」をクリックして開発者登録を行ってください（無料）

ステップ3: 新しいアプリケーションを作成

  • 「New Application」ボタンをクリック
  • アプリケーション名を入力（例: Recipe AI）
  • 説明を入力（例: Recipe suggestion app）
  • 利用規約に同意してSubmit

ステップ4: Access Keyをコピー

作成したアプリケーションの「Access Key」をコピーして、上記の入力欄に貼り付けてください

💡 ヒント: 無料のDeveloperアカウントで1時間あたり50リクエストまで利用可能です
