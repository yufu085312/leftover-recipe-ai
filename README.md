# 冷蔵庫残り物レシピ提案AI

AIが冷蔵庫の残り物から美味しいレシピを提案してくれるWebアプリケーションです。

## 特徴

- 🥬 **食材登録**: 冷蔵庫にある食材を簡単に登録
- ⚙️ **柔軟な制約設定**: 調理時間、難易度、食事タイプ、辛さなどを指定可能
- ✨ **AI生成レシピ**: Google Gemini APIを使用して3〜5種類のユニークなレシピを生成
- 🔄 **瞬時にレシピ修正**: 「もっとヘルシーに」「子供向けに甘く」などの指示でレシピを即座に修正
- 🎨 **モダンなUI**: ダークモード、グラスモーフィズム、スムーズなアニメーション

## セットアップ

### 必要な環境

- Node.js (v18以上推奨)
- npm または yarn

### インストール

1. リポジトリをクローン
```bash
git clone <repository-url>
cd leftover-recipe-ai
```

2. 依存関係をインストール
```bash
npm install
```

3. Gemini APIキーを取得
- [Google AI Studio](https://makersuite.google.com/app/apikey)でAPIキーを取得

4. 開発サーバーを起動
```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開きます。

## 使い方

1. **APIキーの設定**
   - 右上の⚙️ボタンをクリック
   - Gemini APIキーを入力して保存

2. **食材の登録**
   - 冷蔵庫にある食材を入力して追加
   - 複数の食材を登録できます

3. **条件の設定**（オプション）
   - 調理時間、難易度、食事タイプ、辛さなどを設定

4. **レシピ生成**
   - 「レシピを生成」ボタンをクリック
   - AIが3〜5種類のレシピを提案します

5. **レシピの修正**
   - 各レシピの「詳細を見る」をクリック
   - プリセットボタンまたはカスタム指示でレシピを修正

## ビルド

本番環境用にビルドする場合:

```bash
npm run build
```

ビルド結果は `dist/` ディレクトリに出力されます。

## プレビュー

ビルド後のプレビュー:

```bash
npm run preview
```

## 技術スタック

- **フロントエンド**: Vanilla JavaScript + Vite
- **AI**: Google Gemini API
- **スタイリング**: Vanilla CSS
- **ストレージ**: LocalStorage
